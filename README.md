# Management Board UI

A simple Management board app built with Ionic + React + TypeScript + Zustand.  
Fokusnya di UI, interaksi, dan state management tanpa backend.

---
## Setup & Running

### 1. Install dependencies

```bash
npm install
```

### 2. Jalankan development server

```bash
npm start
# atau
ionic serve
```

### 3. Build untuk production

```bash
npm run build
# atau
ionic build
```


## Fitur
Multiple column (To Do, Doing, dll)
Task card (title, desc, assignee, dll)
Cover Image Task
CRUD task
Drag & drop antar column
Filter & search
Checklist + progress bar
Attachment Dummy
Toast notification
Dark mode
Zustand Persist (localStorage)
Add new column
